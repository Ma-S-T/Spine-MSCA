# Spine-MSCA
一个用于腰椎MRI图像分割的标注数据集，包含426名患者的426张腰椎T2矢状位MRI图像，使用线性插值方法统一将分辨率修改为512x512并使用归一化方法将强度统一调整到[0,1],数据格式为pt，每一个数据都包含一张原始图像和标注好的(标注区域为腰椎L1-L5、椎间盘D1-D5以及骶骨S1)标签。
